# Open Source

Webmail is built in the open, on purpose. Email is infrastructure people depend on every
day — it shouldn't be a black box you have to trust blindly. Anyone can read the code,
run it themselves, change it, and see exactly what happens to their mail.

## License

Webmail is licensed under the **GNU General Public License v3 (GPLv3)**. In short:

- You're free to use, modify, and self-host it, for personal or commercial use.
- If you distribute a modified version, it has to stay open under the same license —
  improvements flow back to the community instead of getting locked away.

See the full license text in [`LICENSE`](https://github.com/Yukthi-Systems/Webmail-Docs/blob/main/LICENSE).

## Contributing

Issues and pull requests are welcome across all four repos — UI, API, the RMQ worker,
and the [BIMI API](/docs/bimi). Each has its own contributing notes:

- [UI Contributing guide](/docs/ui/contributing)

Start with an issue if you're planning something non-trivial, so the approach can be
discussed before you put the work in. Or drop into [Discord](https://discord.gg/29zTxvque)
if you'd rather talk it through first.

## Rewards for serious contributors

Significant contributions don't go unnoticed. If you land a meaningful feature, a fix
for something that's been genuinely painful, or a sustained pattern of solid
contributions, reach out — we recognize and reward that kind of work.

:::note
The specifics of the rewards program (criteria, process, how to reach us) aren't
finalized yet — check back here, or ask in GitHub Discussions.
:::
